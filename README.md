# Portfolio Website - Kapu Pranav Adithya Reddy

A modern, dark-themed portfolio website with smooth animations and gradient highlights.

## Features

- Responsive design with dark theme
- Smooth scroll animations
- Interactive certification cards with modal popups
- Contact links (Email, Phone, LinkedIn, GitHub)
- Skills showcase with hover effects

## Adding Your Certificates

Simply drag and drop your certificate images into the `certificates` folder with these names:

- `github-copilot.png` or `github-copilot.jpg`
- `nptel-ml.png` or `nptel-ml.jpg`
- `linguaskill.png` or `linguaskill.jpg`
- `social-internship.png` or `social-internship.jpg`

The website automatically detects PNG, JPG, or JPEG formats.

## How to Use

1. Open `index.html` in your web browser
2. Click on any certification card to view the certificate
3. Customize the content in `index.html` as needed
